
# AI-Assisted Job Offer Analysis System

## Overview
The AI-Assisted Offer Analysis System is a web-based decision support system designed to help users evaluate and compare multiple job offers using structured criteria and weighted scoring.

##  Objective
To provide a transparent, explainable, and structured method for comparing job offers while limiting AI usage strictly to company insight enrichment.

##  AI Usage Scope
AI is used only to fetch structured company insights such as:
- Recent layoffs
- Financial stability
- Growth outlook
- Work-life balance
- Learning exposure

The final decision-making logic is deterministic and implemented using a weighted scoring model.

## Core Features
- Multiple job offer comparison
- Customizable criteria weights
- AI-assisted company insight retrieval
- Deterministic weighted scoring
- Ranked recommendations
- Clear explanation of results
